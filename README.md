# Red de Sint Martinuskerk

Website van **Stichting Onder De Mantel Van Martinus** — een lokaal gedragen initiatief om de Sint Martinuskerk in Oud-Zevenaar te redden van de sloop en om te vormen tot een cultureel hart voor de gemeenschap.

## Over het project

De Sint Martinuskerk is een rijksmonument dat al sinds omstreeks 1345 op een terp aan de dijk in Oud-Zevenaar staat. Na de sluiting in 2021 dreigt het gebouw omgebouwd te worden tot appartementen. Wij geloven dat dit verhaal een ander einde verdient.

Deze website vertelt het verhaal van de kerk, haar geschiedenis van bijna 1000 jaar, de kunstschatten die zij herbergt, en onze visie om haar te transformeren tot een bruisend cultureel centrum.

## Lokaal bekijken

Dit is een statische website (HTML/CSS/JS) zonder build-stap. Open `index.html` in een browser, of gebruik een lokale server:

```bash
# Met Python
python3 -m http.server 8000

# Of met Node.js
npx serve .
```
